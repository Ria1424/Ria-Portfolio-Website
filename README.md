## 🌐 Website Navigation & Page Guide

The portfolio is structured into the following pages, accessible via the navigation bar:

1.  **Home (`index.html`)**
    *   Features a professional summary, profile photo (framed in a thin dark brown border), and direct call-to-action buttons to download the resume or write an email.
    *   Includes an interactive contact form allowing users to send direct messages.
2.  **About Me (`about.html`)**
    *   Contains a detailed biography outlining career goals and academic interests.
    *   Features a structured education timeline displaying Class X (CBSE), Class XII (HSC), and Bachelor of Technology (B.Tech) milestones.
3.  **Experience (`experience.html`)**
    *   Showcases professional work experience, including internship details at **IIT Bombay (Research Intern)** and **Accenture (Associate Software Engineer Intern)**.
    *   Includes a dedicated volunteer registry outlining contributions to the IEEE Computer Society, GirlScript Summer of Code, and Thane Municipal Corporation.
4.  **Projects (`projects.html`)**
    *   Displays academic and independent software projects, labeled as **Solo Passion Projects** or **Associated with BTech course**.
    *   Each card features a tech stack list and dual buttons linking to the GitHub repository and the live hosted application.
5.  **Skills & Achievements (`skills.html`)**
    *   Categorizes technical expertise into Programming Languages, Frameworks, Libraries, Tools, and Core Academic Coursework.
    *   Features an achievements ledger linking to competitive coding profiles (CodeChef Rating: `1867`, DSA Rating: `1703`) and professional certifications from Google Cloud, Coursera, Oracle, Deloitte, NVIDIA, and L&T.
6.  **Publications (`publications.html`)**
    *   Lists published research papers (with links to research sites and official Google Drive verification certificates).
7.  **Interactive CV (`resume.html`)**
    *   A print-optimized curriculum vitae layout highlighting core details for physical delivery or instant PDF exports.

## 🏛️ Design Philosophy & Aesthetics

This portfolio is designed with a curated **"Old Money" / Classic Editorial** aesthetic. It leverages a clean, premium visual language featuring:
*   **Typography**: Playfair Display (Serif headings) and Lora (Serif body) paired with Montserrat for clean UI elements.
*   **Palette**: Sleek dark and warm cream hues (`#0F0E0C` dark charcoal, `#FBF9F4` warm cream, and muted gold accents).
*   **Print Optimization**: Styled with print-specific stylesheets to enable clean, professional exports of the interactive CV directly to physical sheets or PDFs.
*   **Micro-Animations**: Smooth fade-ins, hover states, and responsive transitions.

## 🛠️ Tech Stack

*   **Markup & Templating**: Semantic HTML5, Jinja2 (used during Flask development)
*   **Styling**: Custom CSS3 (utilizing modern custom properties/CSS variables, flexbox, and CSS grid)
*   **Logic & Interactions**: JavaScript (ES6) for navigation toggle, interactive contact forms, and modal logic
*   **Icons & Assets**: FontAwesome 6, Google Fonts
*   **Compilation**: Compiled into static HTML from a Python Flask development setup via a custom crawler script
